Minimal Ghost Toolbar — Ownership Indicator Concepts

The minimal-ghost style strips away the chrome, so the current rotated-diamond caret sits alone above the node and reads as an unrelated glyph. Below are eight quieter ways to express the same "this toolbar belongs to this node" relationship. All variants reuse the node's outline/header color so the cue blends in on both light and dark graph themes.

How to read these mocks: each tile shows the same DPF Viewer node and the same ghost-toolbar row. The only thing that changes is the ownership indicator between them. Concept A reproduces today's behavior as a reference.
